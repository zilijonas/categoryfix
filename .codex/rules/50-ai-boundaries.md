# AI Usage Rules

- AI is optional and must not be a prerequisite for core app value.
- AI must never be the sole basis for an automatic category write.
- AI outputs must be labeled `AI-assisted suggestion` anywhere a merchant can see them.
- Store provenance for AI outputs: model, prompt version, input fields sent, and timestamp.
- Send only the minimum product fields required for the AI task.
- AI should be used for explanation refinement or low-confidence fallback candidate generation only.
- If the AI provider is unavailable, deterministic workflows must continue to operate normally.
- Do not present AI-generated rationale as platform truth; present it as supporting context.
